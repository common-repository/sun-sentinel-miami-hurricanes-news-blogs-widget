== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sun-sentinel-um.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag Miami Hurricanes News and Blog Widget to your Sidebar

== Frequently Asked Questions ==
- How do I upload sun-sentinel-um.php to the /wp-content/plugins/ directory?
- Use your FTP manager, browse to the /wp-content/plugins/ directory of your blog and upload sun-sentinel-um.php to that folder.
